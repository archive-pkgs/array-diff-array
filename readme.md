# array-diff

> Simple difference between two arrays

## Install

```
$ npm install --save array-diff
```


## Usage

```js
var arrayDiff = require('array-diff');

arrayDiff(['apple', 'orange'], ['apple', 'banana']);
//=> ['orange', 'banana']
```


## API

### arrayDiff(input)

#### input

*Required*
Type: `array`
Length: 2
